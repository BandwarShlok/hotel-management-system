# Hotel Management System

## Description
Hotel Management System is a web-based application used to manage hotel operations such as room bookings, customer records, check-in and check-out, and room availability. It helps administrators handle hotel data efficiently while reducing manual work and improving overall management.

## Features
- Room booking management
- Customer record management
- Check-in and check-out system
- Room availability tracking
- Admin management system
- User-friendly interface

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: VS Code, Git, GitHub

## Project Structure
Backend  
Frontend  

## How to Run the Project
1. Open the project folder
2. Go to the Backend directory
3. Run `npm install`
4. Start the server using `node server.js`
